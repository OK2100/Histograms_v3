	###	void addEvent(quint8 chID, quint32 bitset, bool doPrintDecoded=false);

	bitset format:
		[flags 8 bit][charge 13 bit][time 12 bit] - this is 33 bits
		therefore higher bit is thrown out - "time info lost" bit	

		[flags 7 bit][charge 13 bit][time 12 bit] - this is 32 bits





							[	Setup channel window	]
	
	It is created inside ChannelHistWidget's constructor.
	It has such inner values as "nBins_charge", "binWidth_time", etc.. 
	as QString.

	When "Setup channel XX" button is clicked ChannelHistWidget	sets all
	setupWindow's inner values in accordance with current values of 
	corresponding variables.
	
	When some values are changed and "Apply" or "Ok" buttons are clicked
	SetupWindow sends a signal to its ChannelHistWidget. Slots inside 
	ChannelHistWidget handle and apply new settings.


									[	HistData	]
	
	Borders of the histogram are constant, them couldn't be changed after creating 
	histo. They are setted in constructor of HistData class.
